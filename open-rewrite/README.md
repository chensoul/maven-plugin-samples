# Open Rewrite

# Usage

We start out with a Java 8 project, with JUnit 4.

Look at the Demo.java, DemoTest.java and pom.xml

Build the project

```bash
mvn verify
``` 

Run open rewrite

```bash
mvn rewrite:run
``` 

Check git for changes

Build the project to see if it still compiles

```bash
mvn verify
```
